
# Car Speed Prediction using Machine Learning

## Problem
This project predicts the speed of a car based on distance travelled and time taken.

## Tools Used
- Python
- Pandas
- Scikit-learn

## Dataset
The dataset contains three columns:
Distance, Time, Speed

Example:
Distance,Time,Speed
100,2,50
150,3,50
200,4,50

## How to Run

1. Install required libraries:
pip install pandas scikit-learn

2. Run the program:
python speed_prediction.py

## Output
The model predicts car speed based on given distance and time.

Example Output:
Predicted Speed: 50
